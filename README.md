# Christian Wulff — Personal Portfolio

A modern, responsive personal portfolio website built with semantic HTML, CSS, and vanilla JavaScript.

## Structure

```
cw/
├── index.html          # Main page
├── css/
│   ├── variables.css   # Design tokens (colors, spacing, fonts)
│   ├── base.css        # Reset & global styles
│   ├── layout.css      # Page layout & sections
│   ├── components.css  # Reusable UI components
│   └── responsive.css  # Media queries
├── js/
│   └── main.js         # Interactivity & animations
└── README.md
```

## Sections

- **Hero** — Introduction with animated typing effect and stats
- **About** — Bio and personal info
- **Skills** — Expertise cards with progress bars
- **Projects** — Featured work showcase
- **Contact** — Form and social links

## Getting Started

Open `index.html` in your browser, or serve locally:

```bash
npx serve .
```

## Customization

1. Update text content in `index.html`
2. Adjust colors and fonts in `css/variables.css`
3. Replace placeholder project links and social URLs
4. Add your photo in the About section (replace the initials avatar)
